# ng-posix

## Description

ng-posix is a hobby operating system project built from scratch for learning purposes.
It is designed to explore low-level systems programming, including bootloading, kernel development, memory management, and basic hardware interaction on x86 architecture.

## Goals

- Understand the full boot-to-kernel execution flow
- Implement a minimal bootloader in x86 assembly
- Build a freestanding C kernel (no standard library)
- Learn memory management fundamentals (paging, heap, segmentation)
- Experiment with interrupts and hardware abstraction
- Gain familiarity with cross-compilation toolchains and QEMU emulation
